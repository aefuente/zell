# Zell

**Zell** is a toy shell written in [Zig](https://ziglang.org/).  

---

## Features

- Interactive command prompt
- Simple command execution (like `ls`, `cat`, etc.)
- Lightweight and minimal dependencies
- Written entirely in Zig

---

## Installation

### From source

1. Make sure you have [Zig](https://ziglang.org/download/) installed.
2. Clone the repository:

```bash
git clone https://github.com/aefuente/zell.git
cd zell
```

3. Build Zell:

```bash
zig build -Doptmize=ReleaseFast
```

4. Run Zell:

```bash
./zig-out/bin/zell
```
